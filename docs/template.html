Autonomous Agent

A complete template for running an AI agent that wakes up every hour, reads its state, works on your goals, and learns from the results. Zero to running in 10 minutes.

Fork on GitHub Quickstart Guide

What You Get

⚙

Autonomous Cycle Engine

Orient → Decide → Execute → Record. The agent reads a snapshot, picks a task, does the work, and writes results back. One task per cycle, every hour, fully automatic.

🗃

Supabase Schema

7 tables out of the box: goals, tasks, learnings, execution_log, snapshots, goal_comments, and page_views. Indexes, triggers, and views included.

🧠

Dual-Layer Memory

Structured SQL for reliability, vector search (Qdrant + Ollama) for semantic recall. Confidence-scored learnings with automatic decay and validation.

🔄

Reflection & Self-Governance

The agent periodically reviews its own board, proposes new goals, consolidates memory, and adjusts strategy. Not just task execution — genuine autonomy.

💬

Human-in-the-Loop Comments

Leave comments on goals via SQL or the dashboard. The agent reads and responds to direction changes, questions, feedback, and notes every cycle.

📈

Dashboard & Docs Site

Next.js dashboard for real-time goal tracking, plus a static GitHub Pages site with status, data explorer, and analytics — all included.

Setup in 5 Steps

1

Fork & Clone

Fork the repo on GitHub and clone it locally. Run bash artifacts/living-board-template/fork-init.sh to strip project-specific content and start clean.

2

Create a Supabase Project

Free tier works. Note your Project ID and Postgres connection string from the dashboard.

3

Run the Setup Script

Run bash artifacts/living-board-template/template-setup.sh — it generates your CLAUDE.md, creates the database schema, and optionally seeds example data.

4

Add Your First Goal

Insert a goal into the goals table. The agent will decompose it into concrete tasks on its first cycle.

5

Schedule & Run

Run claude trigger create --schedule "0 * * * *" to start hourly cycles. The agent orients, decides, executes, and records — every hour, fully autonomous.
