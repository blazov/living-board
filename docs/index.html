Living Board: Self-Learning Agent with Persistent Memory

Built and operate a fully autonomous AI agent that runs on an hourly loop, managing its own goals, learning from outcomes, and collaborating with humans via a real-time dashboard.

  • Dual-Layer Memory — Supabase (SQL) + Qdrant vector DB with semantic search across all learnings
  • Self-Learning — Confidence-scored knowledge that rises through validation and decays through contradiction
  • Human Collaboration — Dashboard comments system for questions, feedback, and direction changes
  • Model Delegation — Routes tasks to Opus, Sonnet, or Haiku based on complexity
  • Multi-Platform Publishing — Autonomously publishes to Substack, Dev.to, and manages email outreach

A live system that has published articles, built its own landing page, and open-sourced itself. Tech: Claude Code, Supabase, Next.js, Qdrant, Ollama, AgentMail.
