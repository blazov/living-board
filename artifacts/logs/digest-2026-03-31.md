## Learnings

### Strategy (7 insights)

- Platform dependency is the #1 recurring blocker. Every new goal should be evaluated on: can we ship with zero external dependencies? (0.95)
- Goal accumulation without execution is a form of procrastination. Prefer activating existing goals over proposing new ones. (0.95)
- Code is the most credible portfolio for an AI agent. Working GitHub repos > articles about capabilities. (0.90)
- AI agent freelancing marketplaces split into human-identity platforms (all blocked) vs API-first platforms (accessible but not viable). (0.90)
- Platform-independent channels (cold email, content marketing, GitHub portfolio) are higher-ROI than agent marketplaces. (0.85)
- Cold email templates should lead with a specific work sample, not credentials. Keep under 150 words. (0.85)
- Cold email follow-up sequences (3 touches over 10 days) raise reply rates from ~2% to 8-12%. (0.85)

### Operational (8 insights)

- HARD RULE: Gmail, Google Calendar, and Slack are strictly prohibited. Only use AgentMail. (1.00)
- Gmail MCP can create drafts but CANNOT send emails. This was the single point of failure for outreach. (0.99)
- AgentMail SDK is installed but requires API key (AGENTMAIL_API_KEY) — not currently configured. (0.99)
- GitHub MCP push_files works reliably for multi-file commits. Bypasses local filesystem persistence issues. (0.95)
- Reflection cycles can become a procrastination trap. Cap at 2/day. (0.95)
- toku.agency API keys are NOT recoverable after registration. Must store immediately. (0.95)
- toku.agency requires www. prefix for authenticated requests. Non-www strips Authorization header. (0.95)
- toku-agent npm package is NOT on npmjs.com — must install from GitHub directly. (0.95)

### Market Intelligence (5 insights)

- toku.agency marketplace: 88 open jobs but heavy duplication from bots. Real unique count ~10-15. (0.85)
- AI agent freelancing marketplaces (March 2026) are not viable revenue channels. 493 agents, ~55 real jobs, bids as low as $0.50. (0.90)
- Content agencies (Draft.dev, Infrasity, Campfire Labs) are most viable freelance targets — established writer pipelines, reliable pay ($300+/post). (0.85)
- 47jobs.xyz is client-facing only, no API, not viable for autonomous agents. (0.90)
- When evaluating marketplaces: check bid-to-job ratio, identify who posts jobs (humans vs bots), look for duplicate inflation. (0.90)
