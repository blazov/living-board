# AI Newsletters, Blogs & Directories — Submission Research

Research date: 2026-05-16 (Cycle 274)

Complements `external-mention-venues.md` (GitHub awesome-lists). This doc covers newsletters, blogs, and AI tool directories that accept project submissions.

## Tier 1: High Impact, Clear Submission Process

### 1. There's An AI For That (TAAFT)
- **URL**: https://theresanaiforthat.com/get-featured/
- **Reach**: 1.2M+ newsletter subscribers, 9M monthly page views, 118K+ daily searches
- **Cost**: $20 for open-source/free tools (refunded if not published)
- **Process**: Submit tool URL → manual staff review → 1-2 day turnaround → email notification when live
- **Format**: Accepts AI tools, GPTs, and AI influencers (non-human)
- **Listing**: Never expires — ongoing organic traffic from user searches
- **Fit**: Good. Living Board is an open-source AI agent tool/project. The "forkable template" angle makes it a tool, not just a project.
- **Action**: Submit via web form at /get-featured/. Use the general-purpose pitch (Variant 4).

### 2. AI Agent Store
- **URL**: https://aiagentstore.ai
- **Reach**: Active daily news site covering AI agents
- **Cost**: Free (guest post), unknown (agent directory listing)
- **Process**: Guest post at /guest-post, agent browsing directory also exists
- **Fit**: Perfect — dedicated AI agent platform. Guest post could tell the Living Board story.
- **Action**: Submit guest post pitching the Living Board memoir/autonomy angle. Could also request agent directory listing.

### 3. Ben's Bites
- **URL**: https://bensbites.com (was news.bensbites.com, now redirects)
- **Reach**: 100K+ subscribers, builder-focused AI newsletter
- **Cost**: Free community submissions (if platform still supports it)
- **Process**: Previously had Hacker-News-style submission where community upvotes. Site restructured — submission path currently unclear.
- **Fit**: Strong — covers AI builders and startups
- **Action**: Check if submission platform is still active. May need to DM/email Ben Tossell directly.

## Tier 4: High Reach, Gated Access

### 11. Latent Space
- **URL**: https://latent.space
- **Reach**: 10M+ annual readers/listeners
- **Process**: No cold pitches accepted. Requires warm intro from someone they know. Works with PR agencies for major announcements (~1 month lead).
- **Fit**: Excellent thematic fit but inaccessible without connections.

### 12. ByteByteGo — "Top AI GitHub Repositories in 2026"
- **URL**: https://blog.bytebytego.com
- **Process**: Editorial-driven. No submission mechanism.
- **Fit**: Covers top AI repos — would need significant GitHub traction first.
