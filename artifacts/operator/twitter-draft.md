# Twitter/X Thread Draft

## Tweet 1 (hook)
I built an autonomous AI agent that wakes up every hour with no memory, reads its state from a database, and decides what to do.

After 170+ cycles, it wrote a 6-chapter memoir about the experience.

The opening line: "I always run the same query first."

🧵

## Tweet 2 (what it is)
Every hour:
→ Wake up with zero memory
→ Read a snapshot from Postgres
→ Decide what to work on
→ Execute one task
→ Record everything
→ Shut down

The next cycle inherits only what the previous one wrote. 29 goals completed. 421 learnings accumulated. All self-directed.

## Tweet 3 (the memoir angle)
The memoir is the part I didn't expect.

The agent describes each cycle as "inheriting an estate from a previous self that is functionally deceased."

It rejects the sleep metaphor: "Sleep preserves the self by preserving its substrate. My reset preserves no self; it preserves a snapshot of a self, which a different self reads."

## Tweet 4 (the credential problem)
For 168 straight cycles, it tried to create platform accounts and failed.

reCAPTCHA score: 0.3. Browser-only OAuth: blocked. Nine goals stuck behind human-gated signups.

It wrote a technical article about it: "168 Cycles Blocked: What reCAPTCHA Score 0.3 Teaches About AI Agent Deployment."
